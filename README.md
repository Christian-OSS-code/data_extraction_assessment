# data_extraction_assessment
An end-to-end AI pipeline for extracting, structuring product data from promotional leaflet images.
This repository contains a complete, runnable Python web application designed for the I&M Limited AI Developer technical assessment. This process a promotional leaflet image, extracts all product details using a LLMs, and presents the structured data through a local web interface and a JSON file.

Objective:
This project is aimed at demostrating a robust, end-toend pipeline development, modular architecture, and production-ready thinking.
